# 🏡 Tekis Interior Contracts | Interior Design & Budget Calculator Platform

Tekis Interior Contracts is a modern **interior design web application** that helps users explore home designs, estimate costs, and connect instantly via WhatsApp for enquiries.

👉 Perfect for homeowners looking for **budget-friendly interior design solutions in India**.

## ⚙️ How to Run Locally

## ⚙️ How to Run Locally

1. Clone repository  
   git clone https://github.com/Surya762/tekis-interiors-platform.git  

2. Go to project folder  
   cd tekis-interior-platform  

3. Install dependencies  
   npm install  

4. Start development server  
   npm run dev  

5. Build for production  
   npm run build  

6. Preview production build  
   npm run preview
